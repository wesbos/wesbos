Quick 1 minute tip for enabling keyboard tabbing on OSX. Never use that mouse to click "Dont' Save" again.

To enable, Open <code>System Preferences</code> >> <code>Keyboard</code> >> <code>Shortcuts tab</code> and then click the <code>All Controls</code> radio button at the bottom.
